completeRequest

open fun completeRequest(@NonNull intent: Intent, @NonNull pendingRequest: String): BrowserSwitchFinalResult

Completes the browser switch flow and returns a browser switch result if a match is found for the given BrowserSwitchRequest

Return

a BrowserSwitchFinalResult.Success if the browser switch was successfully completed, or BrowserSwitchFinalResult.NoResult if no result can be found for the given pending request String. A BrowserSwitchFinalResult.NoResult will be returned if the user returns to the app without completing the browser switch flow.

Parameters

intent

the intent to return to your application containing a deep link result from the browser flow

pendingRequest

the pending request string returned from BrowserSwitchStartResult.Started via start
